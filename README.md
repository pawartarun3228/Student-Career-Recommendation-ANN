Abstract: This project involves the development of an Artificial Neural Network (ANN) model designed to recommend appropriate career paths for students. The recommendations are based on an analysis of students' academic backgrounds, skills, and interests.

## Files

| File | Description |

|------|-------------|

| career_recommender_1__using_ANN.ipynb | Main Colab notebook |

| career_recommender.csv | Dataset comprising 1,195 student records |

## Tools Utilized

- Programming Language: Python 3
- Libraries: TensorFlow/Keras, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn
- Platforms: Google Colab, Jupyter Notebook

## Model Specifications

- Architecture: Dense(128) → Batch Normalization → Dense(64) → Dropout → Dense(32) → Dense(6)
- Dataset: 810 usable records categorized into 6 career paths
- Evaluation Metrics: Accuracy, Precision, Recall, F1-Score, Confusion Matrix
